The OpenFCA Project comprises a set of tools for Formal Concept Analysis, such as context creation, lattice visualization and attribute exploration.

The SpringGraph component (http://mark-shepherd.com/blog/springgraph-flex-component/) was created by Mark Shepherd and is provided under a separate component license (see springgraph/component_license.htm).

The rest of the project is provided under an MIT-style license (see LICENSE.txt).

The project authors are Paul-Valentin Borza and Ovidiu Sabou.
